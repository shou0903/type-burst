TYPE BURST（タイプバースト）とは

TYPE BURSTは、タイピングと落ちもの連鎖パズルを組み合わせた日本語タイピングゲームです。画面のブロックに書かれた日本語をローマ字で入力すると、そのブロックを爆破できます。同じ色をまとめて消し、落下後の自動連鎖を狙うことでスコアが大きく伸びます。

完全無料・登録不要インストールせず、PCブラウザからすぐに遊べます。
4つの難易度初級・中級・上級・神級から、自分の入力力に合わせて選べます。
詳しいタイピング分析KPM、正確率、苦手キー、左右の手や指の傾向を振り返れます。

遊び方

  1. 消したいブロックの日本語をローマ字で入力します。
  2. 文章を打ち切るとブロックが爆発します。
  3. 同じ色を3個まとめて直接消し、落下後に4個つなげると自動連鎖が起きます。
  4. ゲージが満タンになったらEnterキーでTYPE BURSTを発動し、盤面の下3行を吹き飛ばします。
  5. ブロックが危険ラインを越える前に、連鎖と正確な入力でハイスコアを目指します。

ゲームモード

サバイバル

時間とともにブロックの落下間隔が短くなるエンドレスモードです。初級・中級・上級・神級の4難易度に対応し、世界ランキングへ任意のニックネームで記録を登録できます。

CPU対戦

文章を消して相手へ妨害ブロックを送り、先に相手を追い詰める対戦モードです。CPUの強さは3段階から選べます。

チュートリアルと成長記録

初めてでもルールを順番に練習できます。プレイ後はスコア、KPM、正確率、苦手キーなどを確認し、上達の推移を振り返れます。

よくある質問

TYPE BURST（タイプバースト）とは何ですか？

日本語をローマ字で入力してブロックを爆破し、同じ色をつなげて連鎖を作る無料のタイピングパズルゲームです。

無料で遊べますか？

はい。完全無料で、アカウント登録やインストールも不要です。

スマートフォンで遊べますか？

現在のゲームプレイは、物理キーボードを利用できるPCブラウザに対応しています。スマートフォンではゲーム紹介とランキングを確認できます。

「Type Burst」「タイプバースト」「たいぷばーすと」は同じゲームですか？

はい。いずれも正式名称TYPE BURSTを指します。日本語では「タイプバースト」、ひらがなでは「たいぷばーすと」と読みます。
